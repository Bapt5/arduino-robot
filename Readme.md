# Arduino Robot

* [DHT11 0.4.1](https://www.aranacorp.com/fr/mesure-de-temperature-et-dhumidite-avec-un-capteur-dht11/)

* [RTClib](https://github.com/adafruit/RTClib/)
* [Servo](https://github.com/arduino-libraries/Servo)
* [Simple DHT](https://github.com/winlinvip/SimpleDHT)
* [Tiny Wire M](https://github.com/adafruit/TinyWireM)
